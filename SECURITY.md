# Security Policy

## Geltungsbereich

Diese Richtlinie gilt fuer das gesamte Repository.

## Meldung von Sicherheitsproblemen

- Bitte keine oeffentlichen Issues fuer sensible Sicherheitsluecken anlegen.
- Melden Sie Sicherheitsprobleme vertraulich an die Maintainer.
- Beschreiben Sie Risiko, betroffene Dateien und reproduzierbare Schritte.

## Reaktionsziel

- Erstreaktion: innerhalb von 5 Werktagen
- Bewertung und naechste Schritte: so schnell wie moeglich nach Erstreaktion

## Hinweise

- Keine geheimen Zugangsdaten im Repository ablegen.
- Sensible Daten nur minimiert oder referenziert speichern.
