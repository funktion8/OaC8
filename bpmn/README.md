# BPMN Quellmodelle

Dieser Ordner enthaelt die fachlich verbindlichen BPMN-2.0-Quellmodelle (`*.bpmn`).

## Regeln

- BPMN-Dateien sind die fachliche Prozessquelle.
- Python-Implementierung in `src/business_os/` muss auf diese Modelle referenzierbar sein.
- Mermaid-Diagramme sind zusaetzliche Uebersichten und ersetzen keine BPMN-Quelle.
